## 前端模块设计

### 1️⃣ 仪表盘（Dashboard）

**功能**

- 实时告警计数（按严重度分级：低/中/高）
- 最近 N 条告警流（实时更新）
- 流量速率小图（包数/秒或/分钟，可选）

**实现思路**

- **WebSocket 订阅**后端 `/ws/alerts` 实时推送告警数据
- **组件拆分**
  - `AlertCounter.vue`：不同严重度告警计数
  - `RecentAlerts.vue`：滚动显示最近 N 条告警
  - `TrafficChart.vue`（可选）：显示抓包速率

**数据结构示例**

```json
{
  "timestamp": "2025-10-13T14:00:00",
  "alert_id": 123,
  "severity": "high",
  "src_ip": "192.168.1.1",
  "dst_ip": "10.0.0.1",
  "rule_id": 1,
  "packet_summary": "TCP 80 -> 443",
  "raw_hex": "45000034..."
}
```

### 2️⃣ 告警列表（表格）

**功能**

- 列表显示告警
- 点击查看详情（解析字段 + raw hex + 触发规则）
- 支持搜索和过滤（severity、src_ip、rule_id）

**实现思路**

- **Element UI Table** + **分页**
- **过滤器/搜索栏**：调用后端 `/api/alerts?filter=xxx` 获取数据
- **详情弹窗**：`AlertDetail.vue`，展示 packet_summary、raw_hex、触发规则

---

### 3️⃣ 规则管理

**功能**

- 列表 / 新增 / 编辑 / 删除规则
- **Snort规则导入**：支持导入Snort格式规则文件
- **Snort变量配置**：配置网络环境相关的变量（如$HOME_NET）
- 通过 REST API 与后端交互

**实现思路**

- **规则列表**：`RulesPage.vue`，表格显示所有规则
- **编辑/新增表单**：规则表单弹窗
- **导入功能**：
  - Snort规则导入：文件上传 + 解析
  - JSON规则导入：文本框输入JSON格式规则
  - **Snort变量配置**：配置界面调整变量值
- 调用后端接口：
  - `GET /api/rules` - 获取规则列表
  - `POST /api/rules` - 创建规则
  - `PUT /api/rules/{id}` - 更新规则
  - `DELETE /api/rules/{id}` - 删除规则
  - `POST /api/rules/import-snort` - 导入Snort规则
  - `POST /api/rules/bulk-import-json` - 批量导入JSON规则
  - `GET /api/snort-variables` - 获取Snort变量配置
  - `PUT /api/snort-variables/{name}` - 更新单个变量

**Snort变量配置说明**

Snort规则中常用的变量包括：

- `$HOME_NET`：内部网络范围
- `$EXTERNAL_NET`：外部网络范围
- `$HTTP_SERVERS`：HTTP服务器地址
- `$DNS_SERVERS`：DNS服务器地址
- 等等...

这些变量需要根据实际网络环境配置，默认值只是示例。

### 4️⃣ Packet 查看器（简易）

**功能**

- 展示解析后的字段（IP/TCP/UDP headers）
- 原始十六进制或 base64
- 支持回放（离线 pcap 或选定告警）

**实现思路**

- `PacketViewer.vue`：分两部分
  - `HeaderView`：表格/树状显示解析后的字段
  - `RawHexView`：十六进制/可选 base64
- 回放按钮：调用后端 API `/api/replay`，后端根据告警数据或 pcap 重放包

---

### 5️⃣ 系统设置

**功能**

- 网卡选择
- 抓包参数配置（过滤器、抓包包数、捕获长度）
- 启动 / 停止抓包服务

**实现思路**

- `Settings.vue`：表单选择网卡、抓包参数
- 按钮调用后端：
  - `POST /api/sniffer/start`
  - `POST /api/sniffer/stop`

---

### ⚡ 可选 / 加分项

1. **告警时间线图**：`ECharts` 或 `Chart.js` 展示告警密度随时间变化
2. **pcap 上传 & 离线分析**：前端上传文件 -> 后端解析 -> 返回告警列表
3. **导出告警**：提供 CSV / JSON 导出
4. **地理位置**：调用 GeoIP API，根据 IP 展示位置
